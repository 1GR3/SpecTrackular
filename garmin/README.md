# Garmin Port

This folder contains a Connect IQ watch face port for Garmin Venu 2 / Venu 2 Plus / Venu Sq 2.

Files:
- `manifest.xml` - Connect IQ manifest and supported products
- `monkey.jungle` - minimal project build file
- `resources/strings.xml` - localized strings
- `source/SpecTrackularApp.mc` - SpecTrackular watch face code

The port renders the original analog layout with the color wheel, activity dots, rosette motif, battery, active minutes, heart rate, steps, and date.

Flashlight mode is available with a touch-and-hold gesture. Hold once for a full-white screen, hold again to return to the watch face.
